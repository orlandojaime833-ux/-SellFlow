## 📱 Comandos do Bot

### Para Vendedores

| Comando | Descrição |
|---------|-----------|
| `/start` | Menu principal |
| `/criar_loja` | Criar sua loja |
| `/add Nome Preço` | Adicionar produto (ex: `/add "Curso JS" 49.90`) |
| `/produtos` | Listar seus produtos |
| `/remover N` | Remover produto (ex: `/remover 1`) |
| `/ajuda` | Exibir ajuda |

### Para Compradores

| Comando | Descrição |
|---------|-----------|
| `/start` | Menu principal |
| `/comprar` | Ver produtos à venda |

---

## 🚀 Como Começar

### Para vender (seja um lojista)

1. Abra [@SellFlow](https://t.me/SellFlow)
2. Digite `/criar_loja`
3. Adicione produtos: `/add "Produto" 49.90`
4. Compartilhe seu link: `t.me/SellFlow?start=loja_SEU_ID`

### Para comprar

1. Abra [@SellFlow](https://t.me/SellFlow)
2. Digite `/comprar`
3. Escolha o produto desejado
